# Comparative Vulnerability Assessment of Ubuntu Linux and Metasploitable 2

## Project Overview
This project compared a hardened Ubuntu Linux system against a deliberately vulnerable Metasploitable 2 virtual machine.
## Objectives
- Compare exposed services
- Evaluate attack surface
- Assess security posture differences
## Tools Used
- Nmap
- VirtualBox
- Ubuntu Linux
- Metasploitable 2
## Methodology
1. Network discovery
2. Port scanning
3. Service enumeration
4. Comparative analysis
## Results
| System | Open Ports |
|----------|----------|
| Ubuntu | 22 |
| Metasploitable 2 | Multiple |
## Key Observations
- Ubuntu exposed only SSH
- Metasploitable exposed numerous vulnerable services
- Hardening significantly reduced attack surface
